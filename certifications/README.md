# Certification images

Put your certificate images in this folder so they show up in the Certifications section.

## Required filenames (exact)

- `Power bi certificate.png` (Microsoft Power BI – Infosys Springboard)
- `Nptel cloud computing.png` (Cloud Computing – NPTEL)
- `Responsive web design.png` (Responsive Web Design – FreeCodeCamp)

## Notes

- Use `.png` (recommended). `.jpg` also works if you update the extension in `src/sections/Certifications.jsx`.
- If an image is missing, the card still renders (image simply won’t show).
